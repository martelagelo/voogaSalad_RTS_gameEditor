Detailed Description

This class represents the core functionality of any wizard within the Editor. It requires a concrete implementation of a Controller class which extends it along with an associated FXML file which includes specified fx:id's root, save, and errorMessage. This class gives the class which launches the wizard the ability to modify the error message, set a Consumer lambda function for when the save button is clicked, along with getting the data submitted by the wizard, and the stage held by the wizard such that it can be closed.
