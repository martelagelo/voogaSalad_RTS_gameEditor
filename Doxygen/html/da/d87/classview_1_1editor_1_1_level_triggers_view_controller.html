Detailed Description

This class represents the pane held within a TabPane representing all of the Triggers/Goals associated with a level. It holds a button to add a new Trigger/Goal along with a clickable list of the existing Goals. Both of these elements will spawn a TriggerWizard upon double-click. The publically accessible methods of this class include setting the onButtonClick and the onSelectedClick within the list view. Furthermore, one has the ability to update the list of triggers with a list of TriggerPair elements.
