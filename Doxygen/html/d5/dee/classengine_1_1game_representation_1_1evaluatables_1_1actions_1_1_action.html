Detailed Description

The wrapper for an action class. Actions are a layer of abstraction that was added on top of evaluators in order to make more strict type casting of parameters with evaluators. This allows for specifying complex actions using only string pieces as opposed to necessitating the user to type in long VScript strings for actions.
