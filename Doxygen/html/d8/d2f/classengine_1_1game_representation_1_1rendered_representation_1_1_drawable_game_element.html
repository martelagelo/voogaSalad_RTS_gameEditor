Detailed Description

A game element that is capable of being drawn. Combines a game element state as well as an animation player. Essentially wraps a savable game element state with a drawable wrapper to avoid JavaFX serialization problems.

Author
Zach, Jonathan, Nishad, Rahul, John L., Michael D., Stanley
