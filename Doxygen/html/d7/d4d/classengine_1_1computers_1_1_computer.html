Detailed Description

An class following the strategy pattern to allow for a large variety of computing classes that take in an object of interest and a list of objects to compute against. A computer goes through all the objects in the list of objects to compute against, checks if they meet the criteria, and, if they do, adds them to the element that requires the objects.
