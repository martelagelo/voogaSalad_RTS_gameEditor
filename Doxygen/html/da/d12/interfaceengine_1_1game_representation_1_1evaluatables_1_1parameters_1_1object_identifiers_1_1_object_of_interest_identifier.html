Detailed Description

An interface for a set of classes whose only functionality is to get the objects of interest for creating evaluator parameters. This set of classes was created with the choosing of composition over inheritance for the evaluator parameters. This allows each parameter to act on a wide range of gameElements without requiring a large hierarchy of parameter types.
