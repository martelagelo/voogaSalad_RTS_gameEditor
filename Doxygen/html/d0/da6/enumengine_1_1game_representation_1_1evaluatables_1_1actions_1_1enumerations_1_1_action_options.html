Detailed Description

An enumeration for all the possible action options. The enum provides the java class name of the Action through its getClassString method as well as its prompt Required string inputs from the editor for the constructor are placed in the prompt string as a #

Action types give a string representation of the prompt needed to create the action for the editor. # are used to indicate the location of each string input. Additional enumerations of input possibilities are given through the ActionParameters enumeration.

The format for one of these enumerations is Human-readable action name, action class name, action prompt, list of required parameters
