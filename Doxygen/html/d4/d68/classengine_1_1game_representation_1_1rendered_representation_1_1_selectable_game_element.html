Detailed Description

A wrapper for game elements capable of being selected. Adds a "selected" visual appearance to the appearance defined by the DrawableGameElement and handles animations for actions resulting from being selected.

Author
Jonathan , Steve, Nishad, Rahul, John, Michael D., Zach, Stanley
